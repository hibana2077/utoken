# Toy Experiment Summary

## Goal

This experiment compares two tiny classifiers on a synthetic sequence classification task:

- `baseline`: a 2-block model with a standard second block
- `special`: a 2-block model whose second block uses uDTW-based gating

The dataset is generated by a frozen teacher model that already uses the specialized second block. Because of that, the task favors models that can better reproduce the teacher's inductive bias.

## Command

```bash
./.venv/bin/python toy_exp.py
```

## Run Setting

- Device: `cuda`
- Merge mode: `mul`
- Seed: `7`
- Epochs: `20`
- Train size: `256`
- Validation size: `128`

## Result

Final reported summary from the run:

```text
summary | baseline_best_acc=0.891 baseline_best_loss=0.3050 | special_best_acc=0.930 special_best_loss=0.2331
```

## Short Conclusion

In this toy setup, the specialized uDTW-gated block performs better than the baseline. It reaches higher best validation accuracy (`0.930` vs `0.891`) and lower best validation loss (`0.2331` vs `0.3050`).

This result is consistent with the way the dataset is built: labels come from a teacher that already contains the specialized block, so the specialized student has a structural advantage. The experiment is therefore useful as a sanity check that the uDTW-based block can learn and receive gradients, but it should not be treated as strong evidence of generalization to unrelated real-world tasks.
